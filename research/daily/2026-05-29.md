## Strategies This Pass (ranked by confidence)

### 1. Intraday Momentum — SPY ETF (IM-SPY)
- **Score:** 50 — EXPERIMENTAL
- **Asset class:** US Equities (SPY ETF)
- **Timeframe:** Intraday (signal checked at each HH:00 and HH:30)
- **Core concept:** Compute a "noise boundary" corridor as the 14-day rolling average of absolute intraday returns from the open. When SPY price crosses the upper/lower boundary at any HH:00 or HH:30 mark, enter a directional momentum trade. Exit via a VWAP-anchored trailing stop or forced market close. Position sized to 2% daily vol target with up to 4× leverage.
- **Evidence status:** CLAIMED, UNVERIFIED — SSRN 4824172 (Zarattini, Aziz, Barbon 2024); not peer-reviewed; no independent replication found
- **Source links:**
  - [SSRN 4824172](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=4824172)
  - [SFI No. 24-97](https://www.sfi.ch/en/publications/n-24-97-beat-the-market-an-effective-intraday-momentum-strategy-for-s-p500-etf-spy)
  - [Maróy follow-up SSRN 5095349](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=5095349)
- **Verification level:** No independent replication; community analysis identifies leverage inflation; all primary sources 403'd
- **Analyst note:** The economic mechanism (intraday demand/supply imbalance exceeding noise threshold) is well-grounded and related to Gao et al. (2013). However, the strategy uses up to 4× leverage, and when adjusted to 1×, the Sharpe drops from 1.33 to ~0.399. Max drawdown is not reported, and end-of-day drawdown methodology underreports intraday risk with 4× leverage. The follow-up paper (Maróy 2025, SSRN 5095349) uses further parameter optimization to achieve Sharpe >3, which is a red flag for the original parameter choices. Score: 50 (Experimental).

#### 2. Interpretable Hypothesis-Driven Microstructure (arxiv 2512.12924) ✓ COMPLETED
- **Score:** 52 — EXPERIMENTAL (but primarily a METHODOLOGY PAPER — the strategy has NO edge)
- **Asset class:** US Equities (100-stock universe)
- **Timeframe:** Daily
- **Core concept:** A walk-forward RL framework tests five microstructure signal families (institutional accumulation, flow momentum, mean reversion, breakouts, range-bound value) derived from daily OHLCV data. The paper honestly reports statistically insignificant results (p=0.34, Sharpe 0.33, 0.55% annual return). The contribution is the validation framework methodology, not the five specific signals.
- **Evidence status:** CLAIMED, UNVERIFIED — all arxiv/ResearchGate/mirror sources returned HTTP 403
- **Source links:** arxiv 2512.12924 (HTTP 403), HarbourFront Quant review (HTTP 403)
- **Verification level:** Search engine summaries only; no primary source access
- **Analyst note:** The authors deliberately expose a null result to demonstrate that their walk-forward framework correctly identifies non-working signals. This is commendable methodological honesty. The framework (34 walk-forward periods, strict information set discipline, RL execution, realistic costs) is valuable for other strategy testing. The five OHLCV-based patterns themselves have NO demonstrated edge and should not be traded. Key finding: daily OHLCV data is insufficient for microstructure signal generation — tick or order-flow data is needed.

#### 6. Network Momentum Trend-Following (arxiv 2501.07135) ✓ COMPLETED

- **Score:** 52 — EXPERIMENTAL
- **Asset class:** 28 Futures markets (agriculture, energy, metals, equity indices)
- **Timeframe:** Daily (CTA-style)
- **Core concept:** Augments classical MACD trend signal with cross-market lead-lag detection.
  Two methods — Lévy area (detects linear + nonlinear lead-lag at fixed lags) and Dynamic
  Time Warping (DTW, matches return trajectory shapes) — generate pairwise lead-lag matrices.
  These are combined into a learned graph-adjacency matrix. Markets that systematically follow
  a trending "leader" market receive an early trend signal via graph signal processing. Net
  Sharpe ratios confirmed higher than MACD baseline across all NMM variants. Transaction costs
  explicitly modeled; ensemble reduces costs to ~19% below MACD.
- **Evidence status:** CLAIMED, UNVERIFIED — pre-print only (January 2025); not peer-reviewed;
  all primary sources 403'd
- **Source links:**
  - [arxiv 2501.07135](https://arxiv.org/abs/2501.07135)
  - [Imperial College PDF](https://www.imperial.ac.uk/media/imperial-college/faculty-of-natural-sciences/department-of-mathematics/math-finance/239242288---Linze-Li---LI_LINZE_01869053.pdf)
- **Verification level:** Search summaries only; no peer review; no open code
- **Analyst note:** Methodologically interesting extension of TSMOM. The Lévy area is a
  mathematically sophisticated tool. Key concerns: 28×28 adjacency matrix with learned
  parameters is high-dimensional — overfitting risk. Higher inherent transaction costs than
  baseline MACD. Pre-print only. Specific Sharpe numbers not accessible. Score 52 (Experimental).
  Not an implementation candidate (latent < 60, and no open code).
