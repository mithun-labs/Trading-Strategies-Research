  risk-adjusted net returns improved 38–149%; costs explicitly modeled.
- **Evidence status**: CLAIMED, UNVERIFIED — SFI working papers; neither paper peer-reviewed
  as of 2026-06-12 (3.5 years on SSRN); all metrics from secondary search summaries;
  absolute Sharpe ratio NOT REPORTED in any accessible source
- **Source links**:
  - [SSRN 4280465 — Momentum Without Crashes (2022)](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=4280465) — HTTP 403
  - [SSRN 4955388 — Smoothing Out Momentum and Reversal (2024)](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=4955388) — HTTP 403
- **Verification level**: None — SFI working papers; primary papers blocked; no open code
- **Analyst note**: The most important fact about this pair of papers is that they have
  been on SSRN for 3+ years (2022 paper) without journal publication. The core improvement

## Emerging Patterns

- **Factor timing proliferation, quality varies**: We now have three entries on US equity
  factor timing/rotation: `dynamic-factor-allocation-regime-shu-mulvey` (61, Worth Researching,
  peer-reviewed), `systematic-reversal-industry-momentum` (36, Low Confidence), and now
  `factor-timing-momentum-regime-tai` (34, Low Confidence). The pattern is clear: simple
  momentum-following regime detection on US factors produces statistically marginal improvements
  that don't survive scrutiny once transaction costs and max drawdown are examined.
- **SSRN access remains blocked**: All SSRN direct fetch attempts returned HTTP 403 this run.
  This continues to be the binding constraint on research quality. Secondary search engine
  summaries are the only source for quantitative claims.
- **Factor timing literature consensus**: The Dichtl et al. (2019, FAJ) paper is the
  clearest precedent: factor timing with momentum signals is a "cautionary tale" after
  transaction costs. New papers continue to propose similar approaches with marginal
  improvements, but none from accessible sources demonstrate robust OOS evidence.
- **SFI working paper longevity as a quality signal**: The fractional momentum papers (2022)
  have been on SSRN for 3.5 years without peer review acceptance. This run reinforces the
  observation that SSRN residence time is a useful (inverse) proxy for evidence quality.
  Papers that represent genuine advances typically clear peer review within 1-2 years. Papers
  that persist longer often have methodological issues that reviewers have identified. This
  pattern should be applied consistently in future screening decisions.
- **Absolute Sharpe ratio as a disclosure standard**: Two consecutive strategies this run
  had no accessible absolute Sharpe ratio — only relative improvement claims. This is a
  methodological red flag. Papers that show only "X% better than baseline" without disclosing
  the actual Sharpe allow cherry-picking of bad baselines. Future screening should downgrade
  papers that omit absolute risk-adjusted return figures.
